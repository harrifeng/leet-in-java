* Flatten Binary Tree to Linked List
  + Given a binary tree, flatten it to a linked list in-place. For example, given
    #+begin_example
               1
              / \
             2   5
            / \   \
           3   4   6
    #+end_example
  + The flattened tree should look like:
    #+begin_example
         1
          \
           2
            \
             3
              \
               4
                \
                 5
                  \
                   6
    #+end_example
